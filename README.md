# SaaS User Engagement & Revenue Analysis

## Project Overview

SQL data warehouse analyzing SaaS subscription revenue, user engagement patterns and churn indicators across enterprise accounts.

## Tools Used

- SQL (PostgreSQL/Snowflake/BigQuery compatible)
- Python — data preparation

## Business Questions

- Which subscription tiers generate most monthly recurring revenue?
- Which product features show highest engagement before churn?

## Dataset Source

Simulated SaaS transaction data
